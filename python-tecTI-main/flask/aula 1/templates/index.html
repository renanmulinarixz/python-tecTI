DECORATOR

1 - É uma função que "envolve" outra função para modificar ou estender o seu comportamento sem alterar o código original dela.

  • 2 -
      Reutilização de código: Permite aplicar a mesma lógica (como logs, autenticação ou medição de tempo) em várias funções diferentes.
      Separação de conceitos: Mantém a lógica principal da função limpa, deixando tarefas secundárias para o decorator.
      Controle de acesso: Muito usado para verificar se um usuário está logado antes de permitir a execução de uma ação.

    • 3 - Registro de Rota: Ele avisa ao servidor Flask que aquela função específica deve ser executada sempre que alguém acessar uma URL determinada (ex: o endereço principal /).
